# Expense Tracker with Data Analysis

A command-line expense tracker built using Python, Pandas, and Matplotlib.

## Features

- Add new expenses
- View all expenses
- Analyze expenses by category
- Monthly spending analysis
- Generate Pie Chart, Bar Chart, and Line Chart

## Technologies Used

- Python
- Pandas
- Matplotlib
- CSV

## How to Run

```bash
pip install -r requirements.txt
python Expense_tracker
